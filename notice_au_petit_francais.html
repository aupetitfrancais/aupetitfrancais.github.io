1Présentation générale

La carte numérique d'Au Petit Français est un fichier HTML unique (menu_au_petit_francais_v3.html) qui fonctionne directement dans un navigateur web, sans connexion internet requise pour l'utilisation quotidienne (sauf pour charger les polices Google au premier lancement).

Elle est optimisée pour :

  • 📱 Téléphones (iOS Safari, Android Chrome)
  • 📟 Tablettes (iPad, Android tablette)
  • 💻 Ordinateurs (tous navigateurs modernes)
  • 🖨️ Impression (la notice peut être imprimée via le navigateur)
💡Conseil : Pour un usage en boutique, ouvrez le fichier dans Chrome ou Safari et utilisez l'option "Ajouter à l'écran d'accueil" pour créer une icône comme une vraie application.
2Naviguer dans la carte (pour les clients)

Les clients peuvent parcourir la carte de deux façons :

A
Onglets en haut de l'écran

Appuyez sur un onglet (🥐 Français, 🫓 Colombien, ☕ Boissons, 🍦 Glaces, etc.) pour afficher la rubrique correspondante.

B
Glissement gauche / droite (swipe)

Sur mobile ou tablette, glissez votre doigt horizontalement pour passer d'une rubrique à l'autre.

☕L'onglet Boissons est divisé en 3 sous-sections : Boissons Chaudes / Boissons Froides / Sodas & Eau — faites défiler vers le bas pour tout voir.
3Changer la langue FR / ES
1
Bouton drapeau dans le header

En haut de la page, appuyez sur le bouton 🇪🇸 Español pour basculer toute la carte en espagnol.

2
Retour au français

Appuyez sur 🇫🇷 Français pour revenir en français.

3
Langue par défaut (Admin)

Dans le panneau Admin → onglet ⚙️ Config, vous pouvez définir quelle langue s'affiche au chargement de la page.

⚠️Les traductions en espagnol sont pré-remplies avec des propositions. Vous pouvez les personnaliser dans le panneau Admin pour chaque produit (champs avec bordure dorée = champs espagnol).
4Accès Administrateur

Le bouton "Gérer la carte" est invisible pour les clients. Pour y accéder :

1
Tapez 5 fois sur le logo

Appuyez rapidement 5 fois de suite sur le logo "Au Petit Français" en haut de la page.

2
Fenêtre de mot de passe

Une fenêtre s'ouvre et demande le mot de passe administrateur.

3
Saisissez le mot de passe

Le mot de passe par défaut est : apf2024
Appuyez sur Entrée ou cliquez "Valider".

4
Le bouton ⚙️ apparaît

En bas à droite de l'écran, le bouton "Gérer la carte" est maintenant visible. Appuyez dessus pour ouvrir le panneau admin.

5
Se déconnecter

Dans le panneau admin, appuyez sur 🔒 Déconnexion en haut à droite. Le bouton disparaît immédiatement.

🔐Important : Changez le mot de passe par défaut dès la première utilisation (voir section 10).
5Gérer les produits

Une fois dans le panneau Admin, sélectionnez l'onglet de la rubrique à modifier (🥐 Français, 🫓 Colombien, ☕ B. Chaudes, etc.)

Modifier un produit existant :

①
Nom du produit

Champ du haut (sans bordure) = nom en français. Champ avec bordure dorée = nom en espagnol.

②
Description

Même principe : description FR (sans bordure) puis description ES (bordure dorée).

③
Prix

Entrez le prix en COP (pesos colombiens), sans les points ni les virgules. Exemple : 14000 pour 14 000 COP. Pour un produit "sur demande", mettez 0.

④
Badge

Choisissez un badge dans la liste : ✨ Nouveau / ⭐ Populaire / 🌿 Végé — ou laissez vide.

⑤
Disponibilité

Le toggle "Dispo" permet de griser un produit épuisé sans le supprimer. Le produit apparaîtra barré sur la carte.

Ajouter un nouveau produit :

1
Bouton + Ajouter un produit

En bas de chaque rubrique, appuyez sur ce bouton. Un nouveau produit vide apparaît, que vous pouvez remplir.

Supprimer un produit :

1
Icône 🗑️

Appuyez sur la corbeille en haut à droite de la fiche produit. La suppression est immédiate (dans la session de modification).

⚠️N'oubliez pas ! Toujours appuyer sur 💾 Enregistrer les modifications en bas du panneau admin pour sauvegarder vos changements. Sans ça, tout est perdu à la fermeture.
6Changer l'ordre des articles

Vous pouvez réordonner les produits dans chaque rubrique par simple glisser-déposer :

1
Repérez l'icône ☰

À gauche de chaque fiche produit dans l'admin, l'icône ☰ indique que vous pouvez la déplacer.

2
Cliquez et glissez (ordinateur)

Maintenez le clic sur ☰ et déposez la fiche à la position souhaitée.

3
Sur mobile (tablette)

Appuyez longuement sur la fiche entière et faites-la glisser à la position voulue.

4
Enregistrez

Appuyez sur 💾 Enregistrer pour confirmer le nouvel ordre.

💡L'ordre défini dans l'admin est exactement l'ordre affiché aux clients sur la carte.
7Mettre un produit en avant (⭐ À la une)

Cette fonction permet de mettre en valeur un produit de trois façons simultanées :

  • 🪟 Popup d'accueil : une fenêtre apparaît automatiquement à l'ouverture de la carte (une fois par session)
  • 🎉 Onglet Promos : le produit est automatiquement listé dans les promotions
  • 👨‍🍳 Onglet Chef : le produit apparaît également dans la sélection du chef
1
Ouvrez le panneau Admin

Tapez 5 fois sur le logo → entrez le mot de passe.

2
Choisissez la rubrique

Naviguez vers l'onglet de la catégorie du produit à mettre en avant.

3
Activez le toggle ⭐

Dans la fiche produit, en bas, activez le toggle doré "Mettre en avant". La case devient or quand c'est actif.

4
Enregistrez

Appuyez sur 💾 Enregistrer. Le produit apparaîtra en popup au prochain chargement et dans les onglets Promos & Chef.

💡Vous pouvez mettre plusieurs produits en avant. La popup affiche le premier de la liste. Les autres apparaissent dans les onglets Promos et Chef.
⚠️Un produit marqué "Indisponible" n'apparaîtra pas dans la mise en avant, même si le toggle est activé.

9Rubriques Boissons et Glaces

Boissons (3 sous-catégories) :

  • ☕ Boissons Chaudes — Gérer via Admin → onglet "☕ B. Chaudes"
  • 🥤 Boissons Froides — Gérer via Admin → onglet "🥤 B. Froides"
  • 🫧 Sodas & Eau — Gérer via Admin → onglet "🫧 Sodas"

Les trois sous-catégories s'affichent sur la même page "Boissons", avec des titres séparateurs. L'ordre des articles dans chaque sous-catégorie est modifiable indépendamment.

Glaces :

L'onglet 🍦 Glaces est une rubrique complète et indépendante. Gérez-la via Admin → onglet "🍦 Glaces". Vous pouvez y ajouter tous les parfums, coupes, sorbets et desserts glacés.

➕Ajouter une nouvelle rubrique (ex : Snacks, Pâtisseries…) : cela nécessite une modification du fichier HTML. Contactez le développeur ou signalez-le dans les feedbacks Claude.
10Changer le mot de passe administrateur
1
Ouvrez le panneau Admin

5 taps sur le logo → mot de passe actuel

2
Onglet ⚙️ Config

Appuyez sur l'onglet "⚙️ Config" dans le panneau admin.

3
Section "Mot de passe administrateur"

Entrez le nouveau mot de passe dans le premier champ, confirmez-le dans le second.

4
Cliquez "Mettre à jour"

Le nouveau mot de passe est sauvegardé dans le navigateur.

⚠️Notez bien votre mot de passe ! S'il est perdu, le seul moyen de récupérer l'accès est de modifier le fichier HTML directement (changer la valeur 'apf2024' dans le code).
🔒Le mot de passe est stocké dans le navigateur local (localStorage). Il ne sera pas synchronisé si vous utilisez le fichier sur un autre appareil ou navigateur. Sur un autre appareil, le mot de passe est toujours apf2024 jusqu'à ce que vous le changiez depuis cet appareil.
11Questions fréquentes & dépannage

❓ La carte ne s'affiche pas correctement

Vérifiez que vous utilisez un navigateur moderne (Chrome, Safari, Firefox, Edge). Évitez Internet Explorer.

❓ Les polices de caractères ne s'affichent pas bien

La carte utilise Google Fonts qui nécessite une connexion internet au premier chargement. Sans connexion, des polices de substitution s'affichent.

❓ Mes modifications ne sont pas sauvegardées

Vérifiez que vous avez appuyé sur le bouton 💾 Enregistrer les modifications AVANT de fermer le panneau admin.

❓ Les données ont disparu après un effacement du cache

Les données sont stockées dans le localStorage du navigateur. Si le cache est vidé, les données par défaut sont restaurées. Pour éviter cela, vous pouvez copier le contenu du localStorage (option développeur du navigateur) ou travailler avec un fichier HTML modifié avec vos données intégrées.

❓ Le logo ne s'affiche pas

Le fichier image LOGO_008.png doit se trouver dans le même dossier que le fichier HTML. Si le logo est absent, le nom "Au Petit FRANÇAIS" s'affiche automatiquement à la place.

❓ Comment utiliser la carte sur un iPad en boutique ?

1
Ouvrez le fichier dans Safari
2
Appuyez sur le bouton Partager (icône carré avec flèche)
3
"Sur l'écran d'accueil" — la carte devient une app !

❓ Les animations sont trop lentes sur mon téléphone

Repassez sur le template Standard pour désactiver toutes les animations.
